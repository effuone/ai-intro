To sum up, we derive core principle of autonomous agents:

  • Search for repetitive cycles that need automation
  • Break down cycle into a separate step independent functions
  • Actualize information every month/week/day/hour
  • Build client communication: Telegram bot/Google Calendar/Web-app

Prompt construction != Prompt engineering

Importance of Each Section in Prompt Construction

Section Importance Level Description
Task Mandatory Clearly state the action you want ChatGPT to perform, starting with an action verb.
Context Important Provide relevant background information to constrain possibilities and guide the output.
Exemplars Nice to Have Include examples to improve the quality of the output by providing reference structures.
Persona Nice to Have Specify who ChatGPT should emulate to tailor the response to a particular perspective or expertise.
Format Nice to Have Describe the desired format of the output for clarity and usability.
Tone Nice to Have Indicate the tone of the response to match the desired style and emotional nuance.
